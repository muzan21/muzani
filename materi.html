Materi Pemrograman Web

Beranda Materi Tentang
Web adalah sistem informasi berbasis internet yang diakses menggunakan browser melalui protokol HTTP dan HTTPS.
HTML (HyperText Markup Language) digunakan untuk membuat struktur halaman web seperti heading, paragraf, gambar, dan hyperlink.
Elemen HTML terdiri dari tag pembuka, isi, dan tag penutup. Atribut digunakan untuk memberikan informasi tambahan pada elemen.
CSS (Cascading Style Sheets) digunakan untuk mengatur tampilan website seperti warna, layout, font, dan jarak.
Layout web mengatur posisi elemen menggunakan Flexbox dan Grid agar website tampil responsif di berbagai perangkat.
JavaScript digunakan untuk menambahkan interaktivitas pada website seperti event, manipulasi DOM, dan validasi sederhana.
DOM memungkinkan JavaScript mengakses dan mengubah elemen HTML secara dinamis tanpa database.
Mahasiswa membuat website statis sebagai project akhir menggunakan HTML, CSS, dan JavaScript.
